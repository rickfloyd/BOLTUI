# **App Name**: Quantum CyberVision

## Core Features:

- Markets Dashboard: Real-time data for FX, crypto, metals, and indices from multiple APIs synchronized and displayed in a unified dashboard.
- Chart Engine: Interactive charting tool with drawing, analytics, and indicators, ensuring unique and legal implementation.
- AI Predictive Analysis: Tool that generates predictive analysis and trading signals based on market data.
- Global Dropdown Tabs: Quick-access dropdowns with live previews of top social platforms, corporate apps, and news publications.
- Customizable User Dashboard: Personalized dashboard to organize financial, sports, social, and other feeds, with user preferences stored in Firestore.

## Style Guidelines:

- Background color: Dark gradient from black (#000000) to deep charcoal (#0A0A0F).
- Primary color: Electric purple (#A020F0) to convey a futuristic and energetic feel, ensuring contrast against the dark background.
- Accent color: Cyan (#00FFFF) used for interactive elements and highlights, creating a visually striking contrast.
- Font: 'Orbitron' sans-serif font to maintain a modern and digital cyberpunk-inspired feel.
- Use soft pulse and glow effects on key interactive elements, to make the app more visually appealing and reinforce its theme.
